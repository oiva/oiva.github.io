Windows 7

8.1.2009

Microsoft julkaisee Windows 7:n ilmaisen beta-version 9.1. eli huomenna perjantaina. On jotenkin hämmentävää olla innostunut uudesta Windowsista. Vistaa en onneksi joutunut käyttämään kuin yhdessä työpaikassa, enkä ole harkinnut asentavani sitä omaan koneeseeni.

Windows 7:n luvataan olevan paranneltu Vista. Sen pitäisi olla tehokkaampi, ja kysellä vähemmän käyttäjiä ärsyttäviä turvallisuuskysymyksiä.

Uusista featureista on lista Wikipediassa, joista hauskimmilta näyttävät Aero Peek ja Aero Shake. Peekkiä esitellään tarkemmin tässä videossa.

Tarkempaa tietoa Windows 7:n beta-version hankkimisesta saa Windows Blogista. DVD-image ilmestyy Microsoftin Windows 7-sivulle, josta 2,5 miljoonaa ensimmäistä saavat sen ladata. Päivittäminen onnistuu vain Vista SP1:stä, toinen vaihtoehto on puhdas asennus. Beta-versio lakkaa toimimasta elokuussa.
