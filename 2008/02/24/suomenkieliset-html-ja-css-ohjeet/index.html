Suomenkieliset HTML ja CSS -ohjeet

24.2.2008

Tero kyseli HTML ja CSS -tutoriaalien perään, ja tulin samalla katsastaneeksi suomenkielisten oppaiden tarjonnan.

  • Jukka Korpelan Web-julkaisemisen opas on Kattava opas suurin piirtein kaikkeen aiheeseen liittyvään. "Miten teen Web-sivun" on ihan pätevä.
  • Jukka Tujulan johdanto kotisivujen tekemiseen sisältää hyviä periaatteita, jos jaksaa kahlata "peelosivuista" mouhoamisen läpi.
  • MVnet: Linkkejä kotisivun tekijälle sisältää paljon hyviä linkkejä suomen- ja englanninkielisiin oppaisiin. Vaikuttaisi olevan ajan tasalla.

Bonuksena muutama englanninkielinen sivu:

  • Friendly Bit: Beginner’s guide to CSS. Lisäksi myös standardeista ja hyvistä käytännöistä.
  • CSS Zen Garden avasi aikoinaan ainakin omia silmiäni sille, mitä kaikkea CSS:llä voi tehdä (koskematta sivun sisältöön).
